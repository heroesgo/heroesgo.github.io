状态
string 地图编号
int 地图格位总数
int 玩家数
string[玩家数] 各玩家昵称
int 当前回合数
int 当前玩家
int[玩家数] 各家待援兵力
struct [] 部队信息 ｛
      int 格位编号
      int 玩家
      int 兵力
      int 剩余移动力
      int 剩余攻击机会
｝

int[玩家数] 各玩家最后完成攻击回合编号

指令序列
struct [] 操作回放 ｛
   int 回合数
   int 玩家
   int 操作序号
   int 操作时间戳记
   int  操作类型：增援，行军，攻击，结束
   int  操作格位编号
   int  目标格位编号
   int  操作的兵力数
｝

记录完整的指令序列可用于回放，悔棋；若想记录悔棋操作本身，则扩展操作类型定义，借用剩下字段表达要回到的操作环节，用回合玩家操作编号表示
